# SEO pillar + cluster content map — proposal

**Status:** proposal, awaiting Earth's sign-off on the 5 pillar choices.
**Built:** 2026-05-20 overnight queue item 4.
**Why this exists:** 33 standalone blog posts compete with each other for
ranking and don't accumulate authority. Restructuring them into 5 pillars
+ topical clusters with strong internal linking = 10x the SEO value of
the existing inventory.

## The model (in one paragraph)

A **pillar** is a long (~3000 word) comprehensive page that targets a
head term (high volume, broad intent). A **cluster** is 4-10 shorter
posts that target long-tail queries inside the pillar's topic. Each
cluster post links UP to the pillar. The pillar links DOWN to every
cluster post. Google reads the structure and treats the pillar as
"the authoritative answer" on that topic, which lifts all the cluster
posts. Internal linking is the cheapest SEO move; restructuring 33
existing posts costs nothing but a focused weekend of work.

## Recommended internal-linking changes (sketch)

After the pillars are approved, the actual restructure work would be:

1. **Build 5 new pillar pages** (~3000 words each). Each one is comprehensive on its head term, with a clearly-marked "Read more about X" section linking out to every cluster post.

2. **Edit every existing post** to add a top-of-page banner: "Part of: [Pillar Name] →" linking to the pillar. ~33 small edits, automatable.

3. **Add a "Related in this cluster" footer** to every cluster post linking to 2-3 sibling cluster posts within the same pillar. Boost lateral discovery.

4. **Submit a new sitemap** with the 5 pillar pages at high priority + cluster posts at standard priority. Google re-crawls within ~7 days.

5. **MarketingFooter "Resources" section** — replace today's hand-picked blog links with the 5 pillar links. Surfaces the pillars on every page.

Total effort estimate for the actual restructure (after Earth approves):
- 5 pillar pages × 3 hours each = 15 hours
- 33 post edits × 5 min each = ~3 hours
- Footer + sitemap = 1 hour
- **Total: ~19 hours, one focused weekend**
