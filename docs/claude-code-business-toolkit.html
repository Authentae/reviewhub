Practical "from zero" playbook

If you were starting tomorrow with zero coding skills and one app to build, this is the order I'd run it. Each step is one week unless noted.

  1. Idea + validate

    /product-management:product-brainstorming, then /marketing:competitive-brief, then /sales:account-research on 5 target customers. Decide: am I building the right thing for a real person?

  2. MVP · weeks 2–4

    /init to set up CLAUDE.md, then /workflows:full-stack-feature for each feature. Use /loop overnight.

  3. Polish · week 5

    /security-review, /design:accessibility-review, /design:ux-copy, /marketing:brand-review.

  4. Launch · week 6

    /marketing:campaign-plan, then /marketing:content-creation for blog/email/social. /sales:create-an-asset for the landing page.

  5. Operate + sell · week 7+

    /sales:daily-briefing mornings · /customer-support:ticket-triage when tickets come · /operations:status-report weekly · /finance:financial-statements monthly.

  6. Continuously

    /loop overnight on whatever's on fire. /data:analyze for decisions. /anthropic-skills:skill-creator when you've done a workflow 3+ times.
