## What Wave 4 tests vs Wave 2

If Wave 4 (~8 prospects) hits 70%+ open rate matching Wave 2's
100%, audience-fit is locked. Statistical noise won't be at play.

If Wave 4 hits <50% open rate, Wave 2's 3/3 was a fluke and we
need to re-examine the message + timing.

If Wave 4 hits 70%+ opens AND 1+ replies, the Wave 2 zero-reply
result was sample-size noise (N=3) not a pitch problem. Lower
priority on the audit-preview CTA variant work.

If Wave 4 hits 70%+ opens AND 0 replies, the audit-preview page
itself is the bottleneck. Ship the CTA variant in Week 2 of the
strategic plan.
