# Outreach wave post-mortems

One file per outreach wave. Written within 7 days of the wave's
reply window closing — fresh enough to remember details, far
enough to see signal.

The format is fixed (so the docs compose into a longitudinal
view across waves):

1. **Wave snapshot** — dates, target list, channel, message variant
2. **Result** — opens, audit-URL views, replies, conversions
3. **What worked** — specific things to repeat
4. **What didn't** — specific things to drop
5. **Hypotheses for next wave** — testable predictions

Don't write fluff. Don't write aspirations. Specific, falsifiable
observations only.
