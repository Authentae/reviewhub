# Wave 5 outreach prep — 4-vertical ICP test

**Goal:** test which vertical converts best for ReviewHub by sending the
same product to **5 prospects in each of 4 verticals** (20 emails total).
After 5-7 days you'll have signal on which segment to scale.

**Why this is different from Waves 1-4:** every prior wave was hospitality
only (hotels + cafes). Got 35% audit-view rate, 0 replies. We don't know
if other verticals would reply because we never tested them. This wave
gets that data cheaply.

**Time budget:** ~2 hours total, $0 cost (free tiers of Hunter + Apollo).
Send between **Tuesday 9-11 AM Bangkok** (proven open window) over 2 days.

---

## STEP 1 — Build the prospect list (~30 min, free)

For each of the 4 verticals below, open the Google Maps query, scroll
the results, pick the **5 strongest prospects** matching the filters,
copy into the spreadsheet at the bottom.

### Vertical A — Dental clinics (English-speaking, private)

**Google Maps query:**
```
private dental clinic Bangkok english-speaking
```

**Filters when picking 5:**
- Has English-language Google reviews (not just Thai) → owner speaks English
- 40-200 reviews → big enough to care about reputation, not so big they have an agency
- Star rating 4.3-4.7 → has occasional sub-5 review they wished they'd replied to
- Independent practice (not a chain like Bangkok Smile Group's 30 branches)
- Has a real website (you'll need it for Hunter.io email lookup)

**Why this vertical:** dentists have HIGH per-review revenue impact (a
1-star review costs them ~$500-3000 in lost patients vs. ~$50-200 for a
restaurant). Pre-disposed to care about reply quality. /for-dentists
page is honest about PHI-aware drafts which is their specific concern.

### Vertical B — Specialty cafes / 3rd-wave coffee

**Google Maps query:**
```
specialty coffee Bangkok pour over
```

**Filters when picking 5:**
- Independent (NOT chains like Starbucks, %Arabica, Bluekoff)
- Has English + Thai mixed reviews → caters to expat/tourist + locals
- 100-500 reviews → active enough to be reviewed weekly
- Has Instagram bio with website / contact link
- Café/roastery hybrid → owner-operator typically, decision-maker findable

**Why this vertical:** cafe owners live on their phones, would actually
use the LINE/Telegram alert workflow. Lower price-sensitivity for a
$14/mo tool when their margin is 60-70% on a $4 latte.

### Vertical C — Wellness spas / yoga studios

**Google Maps query:**
```
wellness spa Bangkok thonglor sukhumvit
```

**Filters when picking 5:**
- Independent boutique (NOT Thann Sanctuary chain, NOT Anantara Spa)
- 80-300 reviews
- Has English website AND a Facebook page
- Specific neighborhood (Thonglor, Sukhumvit, Ekkamai) → English-speaking
  clientele → English email outreach works
- Therapist-by-name reviews in their feed → they want the named-therapist
  thank-you flow specifically

**Why this vertical:** spa owners get personally-named reviews ("Khun
Nim was amazing") and feel guilt about not replying to thank specific
therapists. ReviewHub's draft-in-your-voice angle lands hard.

### Vertical D — Muay Thai gyms / boxing fitness

**Google Maps query:**
```
muay thai gym Bangkok english trainers
```

**Filters when picking 5:**
- Independent gyms (NOT Tiger Muay Thai chain Phuket, NOT chain franchises)
- 100-400 reviews
- Caters to foreigners (English reviews + "trial class" mentions)
- Has Instagram with class schedule → owner-marketed
- 4.5+ stars but with occasional 2-3 star "trainer was too intense" type
  reviews — those need careful replies

**Why this vertical:** Muay Thai is a Bangkok-specific differentiator
the verticals page already addresses. Owners often run 1-2 locations
which is the multi-location use case for the Business tier (when Pro/
Business un-gate).

## STEP 2 — Find emails (~45 min, free)

For each of the 20 prospects' websites:

**Hunter.io flow (best first try):**
1. Sign up at hunter.io → confirm email
2. **Domain Search** → paste prospect's website domain (e.g., `chakrabongse.com`)
3. Hunter returns emails associated with that domain ranked by confidence
4. Pick the highest-confidence email — ideal: owner / manager / admin role.
   Fallback: `info@`, `contact@`, `hello@` (lower reply rate but works)
5. Copy email + role into spreadsheet

**Apollo.io flow (try when Hunter fails):**
1. Sign up at apollo.io → free tier 50 credits/mo
2. **People Search** → company name + location filter
3. Returns owner/manager profiles with emails when they're in the DB
4. Apollo is weaker for Thai SMB but try it as Plan B

**Manual fallback (when both fail):**
1. Open the prospect's website → `/contact` or footer
2. Many small businesses just list `info@theirbusiness.com`
3. Use that. Lower reply rate but better than no email.

**Expected outcome:** 14-16 working emails from 20 attempts. Some
prospects will have NO findable email (skip them, find another).

## STEP 3 — Email templates (copy + personalize)

**Universal subject-line pattern (proven 35% open in Wave 4):**
```
[Business name] — quick note about your Google reviews
```

OR if you want sharper testing:
```
Saw [specific thing] in your reviews — quick thought
```
(requires you to actually skim 3 of their reviews first; 5-10 min per
prospect, but doubles open rate)

**Universal body structure:** opening hook with prospect-specific
detail → what you noticed → what ReviewHub does in 2 sentences →
audit URL → soft CTA. ~120 words. Personal, founder-voice, no auto-
template feel.

## STEP 4 — Generate the audit URLs (per prospect)

Before you can paste each `[AUDIT URL]`, you need to create one outbound
audit per prospect in the admin:

1. Open `https://reviewhub.review/outbound-audits` (logged in as admin)
2. Click **+ New audit**
3. Enter: prospect's business name + their Google Place ID (search by
   name in the form)
4. Server generates a share token; click **Copy URL**
5. Paste into your spreadsheet for that prospect

~3 min per audit × 20 prospects = ~1 hour. Yes it's manual; tracking
per-prospect opens via this flow is how we know who's hot vs cold.

## After Wave 5 — measuring signal

**5-7 days post-send, check the admin outreach-stats:**
```
https://reviewhub.review/api/admin/outreach-stats
```
(or via authed Chrome → `/admin/brief` once that page surfaces this)

The numbers you want to see, per vertical:
- **Audit-view rate by vertical** (was 35% in Wave 4 hospitality —
  does dental beat that? does spa under-perform?)
- **Reply rate by vertical** (was 0% — anything > 0% is signal)
- **Specific opens >1 view per prospect** (sign of high interest;
  Chakrabongse hit 14 views in Wave 3 — the warmest signal ever)

**Decision after Wave 5:**
- If ONE vertical beats hospitality's 35% audit-view rate by 10+ points
  → that's where Wave 6 goes (20 more prospects in that vertical)
- If reply rate stays at 0% across all 4 verticals → the bottleneck
  isn't vertical-fit, it's the AUDIT PAGE conversion. Focus next on
  audit page A/B testing rather than more outreach.
- If 1+ replies land → drop everything, personally onboard that
  prospect. First customer is the entire ball game right now.

## Don't do these things

- **Don't send all 20 at once on the same morning** — looks like a blast;
  Gmail's sending reputation takes a hit. Spread: 10 Tuesday, 10
  Wednesday, each in the 9-11 AM Bangkok window.
- **Don't follow up sooner than 5 days** — same wave-4 trap of stacking
  touches too fast.
- **Don't add anyone to a sequence/cadence tool** — this is the manual-
  personal-touch wave specifically, exactly the differentiator vs
  Birdeye/Podium that you can't replicate at automation scale.
- **Don't mark anyone as "Replied"** in the dashboard if they didn't
  actually reply (per memory file
  `feedback_dont_mark_replied_without_evidence.md`).
