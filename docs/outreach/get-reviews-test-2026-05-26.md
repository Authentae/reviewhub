# Get-more-reviews test — 5 prospects (the pivot's first real test)

**What this is:** the first test of the NEW pitch (get more reviews),
replacing the parked reply-first Wave 6. Send 5 emails, see if anyone
bites. If even ONE replies where 5 reply-first waves got 0 → pivot
confirmed.

**Why these 5:** all DENTAL clinics. Reasoning from the 2026-05-26
strategy session:
- Appointment business → they HAVE patient emails (tool needs this)
- Dentists struggle to get reviews (people don't naturally review their
  dentist) → real pain
- Same vertical = clean signal (no vertical confound)
- All have verified emails (from wave-6-prospects.md research)

**No demo artifact yet — on purpose.** We test if the PITCH resonates
first (reply rate). If it does, THEN build the get-reviews demo +
LINE-to-customer. Don't build ahead of proof (the mistake we already
made once).

---

## Send rules

- **From:** `earth.reviewhub@gmail.com` (brand account, NOT personal)
- **When:** any weekday 9-11 AM the prospect's local time
- **How:** Gmail → compose → paste → send. (Earth's hands — sending as
  Earth is the one thing the agent can't do.)
- **Track:** note who you sent to + date in a simple list.
- **Wait 5-7 days.** Don't judge before then.

## What to watch for in the replies

| If a prospect says... | It means... |
|---|---|
| "yes, show me" / "how much?" | **Interest.** Pivot working. Move to a real demo / call. |
| "we already use [tool]" | Competitor intel — ask which, why. |
| "we don't have patient emails" | The email-channel objection → this is where LINE-to-customer would help (build it if this comes up a lot). |
| "not interested" | Fine — but note WHY if they say. |
| total silence (all 5) | Either the pitch still doesn't land, or cold email itself is the broken channel (not the pitch). Then we rethink the CHANNEL, not just the words. |

## After this test

- **1+ reply →** pivot has legs. Build the get-reviews demo + consider
  LINE-to-customer. Send 10 more.
- **0 replies →** the problem isn't the pitch, it's the channel (cold
  email). Time to test a different channel (LINE outreach, warm intros,
  walk-ins) or have the harder "is ReviewHub the vehicle" talk.

The whole point: this is a $0, ~30-minute test that finally points the
RIGHT pitch at the RIGHT people. Whatever happens, we learn something
real — which is more than 5 waves of the wrong pitch gave us.
