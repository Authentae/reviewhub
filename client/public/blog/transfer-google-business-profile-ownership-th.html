วิธีโอนเจ้าของ Google Business Profile (ขั้นตอนจริง)

ใช้เวลาอ่าน 8 นาที ภาษาไทย เผยแพร่ 7 พ.ค. 2026

คู่มือสำหรับเจ้าของร้าน ไม่ใช่ที่ปรึกษา — สถานการณ์จริง: เปลี่ยนเอเจนซี่ ขายร้านให้คนอื่น พนักงานเก่าออกแล้วยังถือ profile ของเรา หรือใช้ Gmail ส่วนตัว มาตั้งแต่ปี 2018 และอยากย้ายเข้า workspace ของบริษัท Google มี 2 เส้นทาง ที่คนสับสนกัน

เส้นทาง 1 — เพิ่ม Manager (ใช้เกือบทุกครั้ง)

Manager ทำอะไรได้บ้าง

✓ แก้ข้อมูลธุรกิจ (เวลาเปิด ที่อยู่ เมนู รูป)

✓ ตอบรีวิว ลบรูปที่ผู้ใช้โพสต์

✓ สร้าง post ดู insights

✓ จัดการคนอื่นที่อยู่ใน profile (เพิ่ม/ลบ manager คนอื่น)

✗ ลบ profile ทั้งอัน

✗ โอนกรรมสิทธิ์เป็นเจ้าของจริง (Primary Owner)

✗ ลบเจ้าของคนเดิมออก

99% ของกรณีที่คุณคิดว่าต้อง "โอนเจ้าของ" จริงๆ คือเพิ่ม Manager — เอเจนซี่ใหม่ ทีมการตลาด คนช่วยตอบรีวิว ทุกคนไม่ต้องการความเป็นเจ้าของแท้จริง พวกเขาต้องการ สิทธิ์เข้าทำงาน

ขั้นตอน — เพิ่ม Manager

  1. เข้า business.google.com (ลงชื่อด้วยบัญชีที่เป็นเจ้าของจริง)
  2. เลือก profile ของร้าน → คลิก "Users" (หรือ "ผู้ใช้")
  3. คลิก "Add" / "เพิ่ม"
  4. ใส่ Gmail ของคนที่จะเพิ่ม → เลือก role: Manager
  5. คลิก Invite → คนนั้นจะได้ email ให้กดยอมรับ

เสร็จแล้ว ใช้เวลา 2 นาที ไม่ต้องรอ 7 วัน ไม่ต้องโอนกรรมสิทธิ์

ทำไมเส้นทางนี้ดีกว่า "โอน": เมื่อคุณเลิกใช้บริการเอเจนซี่ คุณแค่กดลบ Manager ออก profile กลับเป็นของคุณเหมือนเดิม ถ้าคุณ โอน กรรมสิทธิ์ไปแล้ว — เอาคืนยากมาก (ต้องผ่าน reclaim process ที่กินเวลาเป็นเดือน)

เส้นทาง 2 — โอน Primary Ownership (ใช้เมื่อจำเป็นจริงๆ เท่านั้น)

กรณีที่ต้องโอนจริงๆ:

  • ขายร้านให้คนอื่น เปลี่ยนเจ้าของบริษัทเลย
  • เจ้าของคนเดิมเสียชีวิต และมีญาติเข้ามาบริหาร
  • ใช้ Gmail ส่วนตัวมาตลอด อยากย้ายไปบัญชี Workspace ของบริษัท

ถ้าไม่ใช่ 3 กรณีข้างบน ใช้เส้นทาง 1 ดีกว่า

ขั้นตอน — โอน Primary Owner

ขั้นที่ 1: เพิ่มเจ้าของใหม่เป็น Manager ก่อน

ทำตามเส้นทาง 1 ข้างบน — เพิ่มคนที่จะรับเป็น Manager รอเขายอมรับ

ขั้นที่ 2: รอ 7 วัน

Google บังคับช่วงรอ 7 วันก่อนที่ Manager คนใหม่จะถูกเลื่อนเป็น Primary Owner ได้ เป็นกฎความปลอดภัย เพื่อป้องกันการแย่ง profile โดยไม่ตั้งใจ หรือโดยฉ้อโกง

ห้ามข้าม. ระบบไม่มีให้ข้าม ไม่มี exception

ขั้นที่ 3: เปลี่ยน role เป็น Primary Owner

  1. เข้า business.google.com → Users
  2. คลิก Manager คนใหม่ → คลิก Role → เลือก "Primary Owner"
  3. ยืนยัน — คุณจะถูกเปลี่ยนเป็น Owner ทั่วไปอัตโนมัติ

ขั้นที่ 4 (ถ้าต้องการ): ลบตัวเองออก

ถ้าคุณไม่ต้องการเข้าถึงแล้วเลย — Owner คนใหม่ลบคุณได้ ห้ามลบตัวเอง ก่อน Owner คนใหม่เข้าควบคุมเรียบร้อย — เผื่อมีปัญหา จะได้กลับมาแก้ได้

กรณีพิเศษ: เจ้าของเก่า "หายไปแล้ว"

สถานการณ์: คุณเข้าซื้อร้าน เจ้าของคนเดิมคุยไม่รู้เรื่อง / โกรธ / หาตัวไม่เจอ — profile ยังเป็นของเขาบน Google

ทางออก: Reclaim process — Google มีฟอร์มสำหรับธุรกิจที่อ้างว่า "เจ้าของเก่าไม่ตอบ"

  1. เข้าหน้า profile ของร้านใน Google Maps
  2. คลิก "Own this business?" หรือ "Claim this business"
  3. Google แสดงว่า "ธุรกิจนี้มีเจ้าของแล้ว" + ปุ่ม "Request access"
  4. กดปุ่ม → ใส่ข้อมูลของคุณ → submit
  5. Google ส่ง email ไปถามเจ้าของคนเก่าว่ายอมโอนหรือไม่ — รอ 7 วัน
  6. ถ้าเขาไม่ตอบใน 7 วัน — Google จะให้สิทธิ์คุณ claim เอง โดยต้องยืนยันตัวตน (ใบทะเบียนพาณิชย์ ใบกำกับภาษี รูปหน้าร้าน)
  7. ถ้าเจ้าของเดิม ปฏิเสธ — มีกระบวนการอุทธรณ์ ที่ต้องส่งเอกสาร ทางกฎหมาย (เช่น สัญญาซื้อขายธุรกิจ) ที่ support.google.com/business/contact/business_other

ทางลัดที่ไม่ค่อยมีใครพูดถึง

ก่อนเริ่มโอน — backup รีวิวทั้งหมด

ถึงจะหายาก — ในกระบวนการโอน บางครั้ง profile โดน "merge" หรือ "split" โดยไม่ตั้งใจ และรีวิวเก่าหายไป มันมักได้คืน แต่ใช้เวลา 2-4 สัปดาห์ที่ Google support ทำให้

ทางแก้ง่ายๆ: ก่อนเริ่มโอน เข้า profile → Reviews → screenshot ทั้งหน้า หรือใช้เครื่องมือ Google Takeout export ข้อมูล profile ออกมาเก็บไว้ ถ้ารีวิวหาย คุณมีหลักฐานยืนยันให้ support

สรุปสั้นๆ

  1. 99% ของเวลา คุณต้องการ เพิ่ม Manager ไม่ใช่โอนเจ้าของ
  2. โอน Primary Owner ใช้เมื่อขายร้าน / เปลี่ยนนิติบุคคล / ย้าย Workspace
  3. ขั้นตอน: เพิ่มเป็น Manager → รอ 7 วัน → เปลี่ยน role → (อาจ) ลบตัวเอง
  4. ถ้าเจ้าของเก่าหายไป — ใช้ Reclaim process ผ่านปุ่ม "Request access" ในหน้า Maps
  5. Backup รีวิวก่อนเริ่ม — ป้องกันการ merge ที่อาจเกิดขึ้น
